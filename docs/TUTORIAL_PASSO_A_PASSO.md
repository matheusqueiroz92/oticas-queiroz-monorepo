## TUTORIAL 5: Rotina do Caixa

### Cenário
Seu dia completo trabalhando com o caixa.

### Manhã (08:00)

```
🌅 CHEGADA AO TRABALHO

    ↓

1️⃣  Liga computador
    Abre sistema

    ↓

2️⃣  ABRIR CAIXA
    ┌────────────────────────────┐
    │ Menu → Caixa               │
    │ Abrir Caixa                │
    │                            │
    │ Pega troco do cofre:       │
    │ R$ 100,00                  │
    │                            │
    │ Conta as notas:            │
    │ • 2x R$ 50 = R$ 100        │
    │ • 5x R$ 10 = R$  50        │
    │ • 10x R$ 5 = R$  50        │
    │ TOTAL: R$ 200,00           │
    │                            │
    │ Digita no sistema:         │
    │ Valor Inicial: R$ 200,00   │
    │ Confirma                   │
    └────────────────────────────┘

    ↓

3️⃣  Organiza o caixa
    ┌────────────────────────────┐
    │ Separa notas por valor:    │
    │ [R$ 100] [R$ 50] [R$ 20]  │
    │ [R$ 10] [R$ 5] [Moedas]   │
    └────────────────────────────┘

    ↓

✅ PRONTO PARA ATENDER!
```

### Durante o Dia (09:00 - 18:00)

```
☀️ EXPEDIENTE

    ↓

VENDAS ACONTECEM...

09:15 - Venda em dinheiro: R$ 450
    ↓
    Sistema registra AUTOMATICAMENTE ✅
    Caixa agora: R$ 650,00

10:30 - Venda em débito: R$ 320
    ↓
    Sistema registra AUTOMATICAMENTE ✅
    Caixa continua: R$ 650,00
    (débito não é dinheiro físico)

11:45 - Cliente paga dívida: R$ 200 (dinheiro)
    ↓
    Sistema registra AUTOMATICAMENTE ✅
    Caixa agora: R$ 850,00

    ↓

12:00 - HORA DO ALMOÇO
    ↓
    Caixa está com R$ 850,00
    Muito dinheiro!
    
    Faz SANGRIA:
    ┌────────────────────────────┐
    │ Menu → Caixa → Sangria     │
    │ Valor: R$ 600,00           │
    │ Motivo: "Depósito banco"   │
    │ Confirma                   │
    └────────────────────────────┘
    
    Tira R$ 600 do caixa
    Coloca em envelope
    Leva ao banco no almoço
    
    Caixa agora: R$ 250,00

    ↓

13:00-18:00 - TARDE
    Mais vendas...
    Sistema continua registrando...
    
    Caixa final: R$ 820,00
```

### Noite (18:00 - Fechamento)

```

    ↓

    Último cliente saiu

    ↓

    │                            │

    │ R$  20 x 3 = R$  60,00     │
    │ R$  10 x 4 = R$  40,00     │

    ┌────────────────────────────┐
    │ Menu → Caixa               │

    └────────────────────────────┘

    ↓

4️⃣  Sistema mostra resumo
    ┌────────────────────────────┐

    ↓

6️⃣  Sistema compara

não ficou pronto no prazo prometido.

    ↓

    ↓

2️⃣  BUSQUE O PEDIDO
    ┌────────────────────────────┐

    │ Busque pelo nome ou OS     │
    │ Abra o pedido              │
    └────────────────────────────┘

    ↓

3️⃣  VERIFIQUE O STATUS
    ┌────────────────────────────┐
    │ Ainda em produção?         │
    │ → Ligue para laboratório   │
    │                            │
    │ Já está pronto?            │
    │ → Peça desculpas por não   │
    │   ter avisado              │
    │ → Cliente pode buscar JÁ   │
    └────────────────────────────┘

    ↓

4️⃣  RESOLVA COM O CLIENTE
    ┌────────────────────────────┐
    │ Seja honesto sobre situação│
    │ Dê previsão realista       │
    │ Ofereça desconto se atrasar│
    │ muito                      │
    │ Anote no pedido            │
    └────────────────────────────┘

    ↓

5️⃣  DEPOIS: Evite novamente!
    ┌────────────────────────────┐
    │ Sempre dê prazo com folga  │
    │ Acompanhe pedidos no prazo │
    │ Ligue quando ficar pronto  │
    └────────────────────────────┘
```
